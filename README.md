LUMINA

Lumina is a little friend with a module to measure your pulse per minute! It tells you if you're okay, 
if you're very relaxed, or if you're very agitated.

I made this little Tamagotchi pulse meter because I found an oximeter that didn't work at my house,
so I opened it up and tried to make it work with my ESP. I used the TX and it worked, but not in the way I wanted.
I ended up buying an Arduino MAX pulse meter and created this little Tamagotchi, which I named Lumina. 
It's like a little keychain that helps you and measures your pulse and acts based on that.
